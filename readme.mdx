# 🧠 Cursor College — Contributing

Welcome to **Cursor College** — a community-built hub for mastering **Cursor**.

> Built by the community. For the community. Powered by Cursor.

---

## ⚡ Quick Start

1. Fork → Clone → Branch
2. Edit your guide/course
3. Commit: `git commit -m "Add <guide/course/fix>: <short>"`
4. Push & open a PR to `main`

---

## 🤝 What to Contribute

- Guides in `guides/`
- Courses in `courses/`
- Fixes/improvements
- Media in `assets/`

---

## 📁 Structure

- `home.mdx` — landing page
- `guides/` — standalone guides
- `courses/` — courses (flat or module folders)
- `assets/` — images/video per course/guide

---

## ✍️ Content Rules

- Each page is `.mdx`
- Lessons include frontmatter `title`
- Keep it clear, concise, example-first
- Store media in `assets/<your-section>/` and use relative paths

---

## ✅ Pre‑PR Checklist

- [ ] Kebab-case filenames
- [ ] `title` frontmatter on lessons
- [ ] Images/links resolve
- [ ] Spelling/grammar passes

---

## 🔄 Pull Requests

- Clear title and short summary
- Explain why the change helps learners

---

## 🌟 Community

- Contribute only what you have rights to share
- Credit sources when applicable

Thanks for helping grow Cursor College! 🙌
